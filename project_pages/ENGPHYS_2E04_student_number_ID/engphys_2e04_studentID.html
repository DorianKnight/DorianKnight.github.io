Student ID Finite State Machine
Project description and documentation
Overview
I designed and built a finite state machine that displays my student ID on a seven-segment display. This project was the culminating design project in a series of curriculum units that taught me the basics of analog and digital circuits:
  • AC phasors
  • Capacitance and RC circuits
  • Power factor
  • Boolean logic
  • Finite state machines (FSM)
I simulated my digital logic using NI Multisim and built the circuit physically using breadboards, JK flip-flops, primitive logic gates, a seven-segment display and a seven-segment decoder chip.
Video explanation
Purpose
The purpose of the project was to design a finite state machine (FSM) that would iterate through a series of states that mapped to my student ID. Each flip-flop stored an individual bit and the combination of four flip-flops served to represent every individual number in my student ID. Upon each clock cycle, the combinational logic would take the current state as input and output the signals to J and !K pins in order to advance the FSM into the next state.

Special attention was paid to minimize the gate count by Karnaugh mapping both the product of sums (POS) and sum of products (SOP) implementations as well as selectively utilizing De Morganâ€™s law to implement certain sections exclusively using NAND gates.
Outcomes
I successfully implemented my student ID FSM using JK flip-flops and primitive logic gates. The simulated circuit and physical build successfully cycled through all individual ID numbers in the correct order.
